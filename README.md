# PSEARCH


## OUTLINE

Psearch a command line tool for searching for a keyword in multiple files and find the matched lines.

Psearch uses different Interprocess Communication(IPC) methods:
 - fork
 - anonymous pipelines
 - shared memory
 - semaphores
